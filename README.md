# Alma Cosmetics — Backend API

Express + TypeScript. Separate repo from the storefront.

## Run locally

```bash
npm install
cp .env.example .env
npm run dev
```

- Health: `GET http://localhost:4000/health`
- API root: `GET http://localhost:4000/api`

## Build

```bash
npm run build
npm start
```
